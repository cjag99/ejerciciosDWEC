Práctica 04-14 JS Metodos Funcionales 5 - reduce, reduceRight , join

  1. Sumar el total de ventas de un día

    Contexto: Calcular el total de ventas realizadas en un día.

    1. Descripción: Array de ventas [100,200,300,400].

      • Usa Array.reduce() para sumar todas las ventas.
    2. Resultado esperado:
      1000

  2. Concatenar nombres de empleados en orden inverso

    Contexto: Concatenar nombres en orden inverso para un informe.

    1. Descripción: Array de empleados ["Juan","María","Pedro","Luis"].

      • Usa Array.reduceRight() para concatenar en orden inverso.
    2. Resultado esperado:
      "Luis Pedro María Juan"

  3. Contar la cantidad de productos en un inventario

    Contexto: Calcular el total de productos.

    1. Descripción: Array de cantidades [5,10,20,15].

      • Usa Array.reduce() para sumar todas las cantidades.
    2. Resultado esperado:
      50

  4. Convertir una lista de palabras en una oración

    Contexto: Unir palabras en una sola cadena.

    1. Descripción: Array ["JavaScript","es","increíble"].

      • Usa Array.join() para unir con espacios.
    2. Resultado esperado:
      "JavaScript es increíble"

  5. Convertir un array de números a una cadena

    Contexto: Convertir números a cadena separada por comas.

    1. Descripción: Array [1,2,3,4,5].

      • Usa Array.toString() para convertir a cadena.
    2. Resultado esperado:
      "1,2,3,4,5"

  6. Mostrar una lista de precios en formato local

    Contexto: Mostrar precios en formato local con símbolo de moneda.

    1. Descripción: Array [1000,2000,3000].

      • Usa Array.toLocaleString() para convertir a formato local.
    2. Resultado esperado:
      "$1,000.00,$2,000.00,$3,000.00"

  7. Calcular el total de ingresos de un mes

    Contexto: Sumar ingresos semanales.

    1. Descripción: Array [500,1000,1500,2000].

      • Usa Array.reduce() para calcular el total.
    2. Resultado esperado:
      5000

  8. Crear un resumen de compras en orden inverso

    Contexto: Listar productos en orden inverso.

    1. Descripción: Array ["Manzana","Naranja","Plátano"].

      • Usa Array.reduceRight() para concatenar en orden inverso.
    2. Resultado esperado:
      "Plátano, Naranja, Manzana"

  9. Generar una cadena de números de un array

    Contexto: Convertir una lista de números en una cadena para fines de visualización.

    1. Descripción: Array de números [10,20,30,40].

      • Usa Array.toString() para convertirlo en una cadena separada por comas.
    2. Resultado esperado:

  10. Mostrar precios en diferentes formatos locales

    Contexto: Mostrar precios en dólares y euros usando formato local.

    1. Descripción: Array de precios [1000,1500,2000].

      • Usa Array.toLocaleString() para convertir los precios en formato de EE.UU. y Europa.
    2. Resultado esperado:
