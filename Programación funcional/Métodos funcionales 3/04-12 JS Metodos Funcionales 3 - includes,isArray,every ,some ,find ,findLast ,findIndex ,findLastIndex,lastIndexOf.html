Práctica 04-12 JS Metodos Funcionales 3 - includes,isArray,every ,some ,find ,findLast ,findIndex ,findLastIndex,lastIndexOf

  1. Verificación de productos en un inventario

    Contexto: Tienes un inventario de productos y quieres verificar si un producto específico está en stock.

    1. Descripción: Array con productos ["manzana","naranja","plátano","pera"].

      • Usa Array.includes() para verificar si "naranja" está en el inventario.
      • Usa Array.includes() desde índice 2 para comprobar si "mango" está en el inventario.
    2. Resultado esperado:
      true // "naranja" está en el inventario
      false // "mango" no está en el inventario desde el índice 2

  2. Verificar si una variable es un array

    Contexto: Necesitas asegurarte de que ciertas variables sean arrays.

    1. Descripción: Variable data que puede ser array o no.

      • Usa Array.isArray() para verificar si es un array.
      • Si es array, recórrelo y muestra los elementos; si no, muestra error.
    2. Resultado esperado:
      const data = [1,2,3];
      Array.isArray(data); // true

  3. Validación de formularios

    Contexto: Verificar si todos los campos están completos.

    1. Descripción: Array de campos [true,true,false,true].

      • Usa Array.every() para comprobar si todos están completos.
      • Si alguno está vacío, muestra advertencia.
    2. Resultado esperado:
      const campos = [true,true,false,true];
      campos.every(campo => campo); // false

  4. Verificación de permisos de usuarios

    Contexto: Comprobar si al menos un usuario tiene permisos.

    1. Descripción: Array de permisos [false,false,true,false].

      • Usa Array.some() para verificar si alguno tiene permisos.
    2. Resultado esperado:
      const permisos = [false,false,true,false];
      permisos.some(permiso => permiso); // true

  5. Encontrar el primer producto con stock bajo

    Contexto: Buscar el primer producto con stock menor a 10.

    1. Descripción: Array de stock [15,7,25,5].

      • Usa Array.find() para encontrar el primero < 10.
    2. Resultado esperado:
      const stock = [15,7,25,5];
      stock.find(cantidad => cantidad < 10); // 7

  6. Encontrar el último pedido urgente

    Contexto: Buscar el último pedido con prioridad "alta".

    1. Descripción: Array de prioridades ["baja","media","alta","baja"].

      • Usa Array.findLast() para encontrar el último "alta".
    2. Resultado esperado:
      const prioridades = ["baja","media","alta","baja"];
      prioridades.findLast(p => p==="alta"); // "alta"

  7. Encontrar el índice del primer producto en oferta

    Contexto: Buscar el índice del primer producto < 50.

    1. Descripción: Array de precios [100,50,30,150].

      • Usa Array.findIndex() para encontrar el índice.
    2. Resultado esperado:
      const precios = [100,50,30,150];
      precios.findIndex(p => p < 50); // 2

  8. Encontrar el índice del último cliente VIP

    Contexto: Necesitas encontrar el índice del último cliente VIP en una lista.

    1. Descripción: Array de clientes ["regular","VIP","regular","VIP"].

      • Usa Array.findLastIndex() para localizar el último "VIP".
    2. Resultado esperado:
      const clientes = ["regular","VIP","regular","VIP"];
      clientes.findLastIndex(c => c==="VIP"); // 3

  9. Búsqueda del índice de un producto específico

    Contexto: Encontrar la posición de un producto en la lista.

    1. Descripción: Array de productos ["manzana","naranja","plátano"].

      • Usa Array.indexOf() para localizar "plátano".
      • Usa Array.lastIndexOf() para localizar el último "manzana".
    2. Resultado esperado:
      const productos = ["manzana","naranja","plátano"];
      productos.indexOf("plátano"); // 2
      productos.lastIndexOf("manzana"); // 0

  10. Validación de usuarios con un tipo específico

    Contexto: Verificar si existe un administrador en la lista de usuarios.

    1. Descripción: Array de roles ["usuario","usuario","admin","moderador"].

      • Usa Array.includes() para comprobar si hay un "admin".
      • Usa Array.some() para verificar si al menos uno es "admin".
    2. Resultado esperado:
      const roles = ["usuario","usuario","admin","moderador"];
      roles.includes("admin"); // true
      roles.some(r => r==="admin"); // true
