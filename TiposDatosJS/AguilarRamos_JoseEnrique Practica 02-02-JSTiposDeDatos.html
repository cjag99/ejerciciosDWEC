Práctica 02-02: Tipos de datos en JS


  1. ¿Qué etiqueta se utiliza para insertar código JavaScript en HTML?

    2. La etiqueta script.

  2. ¿Qué ventajas tiene incrustar el código JavaScript en un archivo externo?

    3. El poder separar el código funcional de la estructura de la web.

  3. ¿Qué extensión suelen tener los archivos que contienen código JavaScript?

    4. La extensión .js.

  4. ¿Cómo se referencia un archivo .js en HTML?

    5. Indicando la ruta del archivo, ya sea mediante la ruta absoluta o la relativa.

  5. ¿Qué debe hacer si el archivo JavaScript se encuentra en otro directorio?

    6. Usar la ruta absoluta.

  6. ¿Qué etiqueta se usa para mostrar un mensaje si el navegador no soporta JavaScript?

    7. La etiqueta <noscript>.

  7. ¿Qué diferencia hay al insertar JavaScript en XHTML?

    8. Tienes que encapsular el código JavaScript en una etiqueta CDATA porque XML es intolerantes a los elementos que no comienzan por < y a las entidades HTML que no comienzan por &.

  8. ¿Qué es la ofuscación de código y por qué se utiliza?

    9. Es una técnica para hacer menos legible el código. Se usa porque el código JavaScript es visible para todo el mundo y, por tanto susceptible de ser robado.

  9. ¿Cómo se llama el lenguaje que se menciona en el texto?

    10. JavaScript.

  10. ¿Qué nombre le dio Microsoft a JavaScript?

    11. JScript.

  11. ¿Por qué Microsoft usó el nombre Jscript?

    12. Para evitar problemas relacionados con la marca (ni tenían derechos sobre el nombre JavaScript, y podrían tener problemas por la parte de "Java").

  12. ¿Qué problema surgió con la versión de JScript de Microsoft?

    13. Su versión de JScript tenía incompatibilidades en varios navegadores.

  13. ¿Qué es el DOM y por qué fue diseñado?

    14. El DOM (Modelo de Objetos del Documento) es una representación estructurada de un documento HTML o XML que permite a los lenguajes de programación (como JavaScript) acceder y manipular el contenido, la estructura y el estilo del documento, y fue creado por el W3C justamente para evitar las incompatibilidades que tenía JScript entre navegadores.

  14. ¿Qué navegadores incorporaron el DOM desde su primera versión?

    15. Internet Explorer 6, Netscape Navigator, Opera 7 y Mozilla Firefox.

  15. ¿Qué estándar regula el lenguaje JavaScript desde 1997?

    16. El estándar ECMA.

  16. ¿Qué documento detalla las especificaciones del estándar ECMA?

    17. El documento ECMA-262.

  17. ¿Qué relación tiene JavaScript con Java?

    18. No tienen relación alguna, pese a que JavaScript adopte nombres y convenciones similares.

  18. ¿Quién desarrolló originalmente JavaScript y con qué nombre?

    19. JavaScript fue originalmente desarrollado por Brendan Eich, de Netscape, con el nombre de Mocha.
